Di dunia data, web search dan web mining adalah dua bidang yang saling terhubung. Web search fokus pada cara menemukan informasi yang relevan di internet. Proses ini dimulai dengan web crawling, di mana program khusus (juga disebut spider atau crawler) menjelajahi internet untuk mengumpulkan data dari halaman web, kemudian menyimpannya dalam index. Ketika pengguna memasukkan kata kunci, algoritma akan mencocokkannya dengan data dalam index dan menampilkan hasil yang paling relevan.
